# Document.breakApart()

## Availability

Flash MX 2004.

## Usage

Document.breakApart()

## Parameters

None.

## Returns

Nothing.

### Type

```typescript
void
```

## Description

Method; Performs a break-apart operation on the current selection.

## Example

The following example breaks apart the current selection:

```javascript
fl.getDocumentDOM().breakApart();
```
