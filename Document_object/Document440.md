# Document.selectNone()

## Availability

Flash MX 2004.

## Usage

Document.selectNone()

## Parameters

None.

## Returns

Nothing.

### Type

```typescript
void
```

## Description

Method; Deselects any selected items.

## Example

The following example deselects any items that are selected:

```javascript
fl.getDocumentDOM().selectNone();
```
