# Document.unGroup()

## Availability

Flash MX 2004.

## Usage

Document.unGroup()

## Parameters

None.

## Returns

Nothing.

### Type

```typescript
void
```

## Description

Method; Ungroups the current selection.

## Example

The following example ungroups the elements in the current selection:

```javascript
fl.getDocumentDOM().unGroup();
```
