# Document.width

## Availability

Flash MX 2004.

## Usage

Document.width

## Description

Property; `integer`; The width of the document (Stage) in pixels.

## Example

The following example sets the width of the Stage to 400 pixels.

```javascript
fl.getDocumentDOM().width = 400;
```
