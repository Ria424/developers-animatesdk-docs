# Document.height

## Availability

Flash MX 2004.

## Usage

Document.height

## Description

Property; `integer`; Specifies the height of the document (Stage) in pixels.

## Example

The following example sets the height of the Stage to 400 pixels:

```javascript
fl.getDocumentDOM().height = 400;
```
