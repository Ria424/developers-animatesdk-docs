## Element.transformX

#### Availability

Flash CS3 Professional.

#### Usage

*element.transformX*

#### Description

Property; `number`; a floating-point number that specifies the x value of the selected elementâ€™s transformation point, within the coordinate system of the element's parent. Setting this property to a new value moves the element. By contrast, the [Element.setTransformationPoint()](../Element_object/Element19.md) method moves the transformation point but does not move the element.
