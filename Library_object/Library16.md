# Library.selectNone()

## Availability

Flash MX 2004.

## Usage

Library.selectNone()

## Parameters

None.

## Returns

Nothing.

### Type

```typescript
void
```

## Description

Method; Deselects all the library items.

## Example

The following examples deselect all the items in the library:

```javascript
fl.getDocumentDOM().library.selectNone();
fl.getDocumentDOM().library.selectAll(false);
```
