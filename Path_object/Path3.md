# Path.clear()

## Availability

Flash MX 2004.

## Usage

path.clear()

## Parameters

None.

## Returns

Nothing.

### Type

```typescript
void
```

## Description

Method; removes all points from the path.

## Example

The following example removes all points from a path stored in the myPath variable:

```javascript
var myPath = fl.drawingLayer.newPath();
myPath.clear();
```
