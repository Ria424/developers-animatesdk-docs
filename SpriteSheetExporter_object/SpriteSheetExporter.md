# SpriteSheetExporter.addBitmap()

## Availability

Flash Pro CS6.

## Usage

SpriteSheetExporter.addBitmap(bitmap)

## Parameters

### **bitmap**

#### Type

```typescript
BitmapItem | Bitmap
```

#### Description

The BitmapItem or Bitmap to include in the sprite sheet.

## Returns

Nothing.

## Description

Method; Adds the specified bitmap or bitmapItem to the sprite sheet object.
