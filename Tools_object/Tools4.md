# Tools.getKeyDown()

## Availability

Flash MX 2004.

## Usage

Tools.getKeyDown()

## Parameters

None.

## Returns

`integer`; The integer value of the key.

## Description

Method; Returns the most recently pressed key.

## Example

The following example displays the integer value of the most recently pressed key:

```javascript
var theKey = fl.tools.getKeyDown();
fl.trace(theKey);
```
