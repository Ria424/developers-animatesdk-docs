## XMLUI.accept()

#### Availability

Flash MX 2004.

#### Usage

XMLUI.accept()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; Closes the current XMLUI dialog box with an accept state, which is equivalent to the user clicking the OK button.
